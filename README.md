# Notes App with Search & Tags

A simple notes management application built with **React** as a learning project to practice component based development and state management.

The app allows users to create, organize, and search notes efficiently.

---

## Features
- Create, edit, and delete notes
- Add tags to notes
- Filter notes by tag
- Search notes by text
- Optional persistence using `localStorage`

---

## Skills Demonstrated
- React component structure
- Props drilling and state handling
- Conditional rendering
- Filtering and search logic
- CSS layout and styling
- Problem solving through CRUD operations

---

## Tools Used
- React
- JavaScript
- CSS

---

## Project Outcome
Built a functional notes application that demonstrates core software development concepts such as CRUD logic, component communication, and data filtering. The project strengthened problem solving skills and practical understanding of React application structure.
